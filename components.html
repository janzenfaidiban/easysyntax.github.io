Getting Started

EasySyntax
UI Components

Design system berbasis Neo-Brutalism untuk platform EasySyntax. Dibangun di atas Tailwind CSS dengan gaya yang tegas, bold, dan fungsional.

Setup & Installation

Tambahkan dependency berikut di head HTML kamu:

<!-- Google Fonts: Plus Jakarta Sans --> <link href="https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@400;600;700;800&display=swap" rel="stylesheet"> <!-- Tailwind CSS CDN --> <script src="https://cdn.tailwindcss.com"></script> <!-- Tailwind Config --> <script> tailwind.config = { theme: { extend: { fontFamily: { sans: ['"Plus Jakarta Sans"', 'sans-serif'] }, colors: { brandBlue: '#3B82F6', brandBlueDark: '#1D4ED8' } }} } </script> <!-- Global Utility CSS --> <style> .btn-flat { position: relative; transition: all 0.08s ease; border-bottom-width: 6px; } .btn-flat:active { transform: translateY(4px); border-bottom-width: 2px; } .card-flat { border-bottom-width: 8px; } </style>
